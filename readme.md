# frontend-test-task

## Установка webpack и webpack-cli (необходимо для сборки)

npm i -D webpack webpack-cli

## Сборка

### Сборка на продакшн

npm run build

### Сборка не сжатая версия (для разработчика)

npm run dev

### Сборка для разработчика с отслеживанием изменений (--watch)

npm run watch
